# open-source-project
Sejong Uni. Open Source Project

###### Controls
- Arrow keys('LEFT', 'RIGHT') to move
- 'SPACE' key to jump
- 's' and 'd' keys to attack

###### Tools & Assets
- [Python](https://www.python.org/)
- [PyGame](https://www.pygame.org/)
